Description List

h1-h6
These tags are used for headings. h1 is the largest heading and h6 is the smallest.
p
The paragraph tag is used to write blocks of text.
a
The anchor tag creates hyperlinks to connect pages or external websites.
img
The image tag is used to display pictures on a webpage.
ul and li
The unordered list and list item tags are used to create bullet point lists.
